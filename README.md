Is You README Ok?
===

`README.ok.md` 파일을 다음과 같이 작성합시다.

```
%% insert templates/hello.c
```

그리고 `templates/hello.c`라는 파일이 다음과 같이 작성되어있다면,

```

#include<stdio.h>

int main() {
	printf("Hello Worldn");
	return 0;
}
```
